# 💼 Hoja de Vida — Juan de Dios Posada Patiño

Este proyecto representa mi **hoja de vida digital (CV web)**, desarrollada completamente con **HTML semántico, CSS estructurado bajo la metodología BEM**, y un pequeño script en JavaScript para manejar el modo claro/oscuro.

El objetivo principal es mostrar mis habilidades técnicas como **Desarrollador de Software** y mi capacidad para construir interfaces limpias, accesibles y escalables siguiendo buenas prácticas de **front-end moderno**.

---

## 🧩 Características principales

- ✅ **Estructura semántica en HTML5** para mejorar legibilidad, accesibilidad y SEO.  
- 🎨 **CSS modular con BEM** (Block Element Modifier), garantizando mantenibilidad y escalabilidad del código.  
- 🌓 **Modo claro / oscuro** persistente mediante `localStorage`.  
- 🧠 Diseño adaptable y responsivo (desktop y mobile).  
- 🧩 Implementación limpia sin frameworks externos (solo HTML, CSS y JS puro).  

---

## 🧠 Tecnologías utilizadas

| Área | Tecnologías |
|------|--------------|
| **Front-End** | HTML5, CSS3 (BEM), JavaScript ES6 |
